<h3>Запуск проекта</h3>
<p>Сначало установить все зависимости<p>

```bash
yarn install
```

<h2>Для запуска нужно два окна консоли</h2>
  
<p>1) Запустить компиляцию из Typescript в Javascript<p>
  
```bash
yarn build:watch
```
<p>2) Запустить проект и наблюдатель nodemon<p>
  
```bash
yarn dev
```
